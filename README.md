# Web à Québec 2013

> Temporary splash page!

## Quick start

    bundle install
    bundle exec jekyll serve --watch
    open http://localhost:4000/

## Deployment

Setup :

    git remote add heroku git@heroku.com:webaquebec2014-splash.git

Deployment :

    git push origin gh-pages
